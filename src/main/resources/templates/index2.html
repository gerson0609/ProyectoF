Manual de Usuario - Bloc de Notas

1. Registro e Inicio de Sesión

Para comenzar a usar el bloc de notas, primero debes registrarte proporcionando un nombre de usuario y una contraseña segura. Después del registro, puedes iniciar sesión utilizando tus credenciales.

2. Creación de Tareas

Una vez que hayas iniciado sesión, puedes crear nuevas tareas especificando un título, una descripción, una fecha de vencimiento y una prioridad. Utiliza la opción de "Crear Tarea" para agregar nuevas entradas a tu bloc de notas.

3. Listado de Tareas

Puedes ver la lista de todas tus tareas utilizando la opción de "Listar Tareas". Esto te mostrará un resumen de todas las tareas que has creado.

4. Actualización de Tareas

Si necesitas realizar cambios en alguna tarea existente, utiliza la opción de "Actualizar Tarea". Puedes modificar el título, la descripción, la fecha de vencimiento y la prioridad de la tarea.

5. Eliminación de Tareas

Para eliminar una tarea, selecciona la opción "Eliminar Tarea" y sigue las instrucciones. Ten en cuenta que esta acción es irreversible.

6. Filtros y Ordenación de Tareas

La aplicación proporciona funcionalidades de filtrado y ordenación para que puedas ver tus tareas de acuerdo a diferentes criterios, como prioridad o fecha de vencimiento.

¡Disfruta usando tu bloc de notas!
